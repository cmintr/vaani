# 60-second offline demo script

1. Put the phone in airplane mode.
2. Launch Vaani from a fresh local-ready state.
3. Show model status: on-device brain ready.
4. Speak: "Remind me at five pm to test the demo."
5. Show the reminder/tool side effect.
6. Ask one memory-backed question that retrieves a stored fact.
7. Ask a cloud-only web-search request and show the graceful deferred response.
8. End by showing no network required for the local turn.

## What this demonstrates

- Local model is installed and ready.
- Voice-command harness path reaches tool execution.
- Memory recall is local.
- Cloud-only tools do not silently fake offline capability.
